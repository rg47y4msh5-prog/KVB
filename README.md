# KVB — Klein Vaarbewijs oefenmateriaal

Statische site met interactieve oefenexamens en visuele naslag voor het CBR Klein Vaarbewijs 1 en 2.
Live: https://rg47y4msh5-prog.github.io/KVB/

## Inhoud

| Bestand | Wat |
|---|---|
| `index.html` | landingspagina |
| `kvb1-oefenbank.html` | KVB1 — 207 vragen over 40 slots, examensimulatie (80 punten, cesuur 56) + leermodus |
| `kvb2-oefenbank.html` | KVB2 — 108 vragen over 27 slots, 23 meerkeuze + 4 invulvragen met marge-validatie, 90 min |
| `kvb2-visuals-navigatie.html` | visuele naslag les 5–6 |
| `kvb2-visuals-betonning-getijden.html` | visuele naslag les 7 |
| `kvb2-visuals-srw.html` | visuele naslag les 8 |
| `kvb1-assets.html` | KVB1-assetbibliotheek — galerij met 213 ingebedde SVG's |
| `kvb2-assets.html` | KVB2-assetbibliotheek — galerij met 85 ingebedde SVG's |

Alle HTML-bestanden zijn standalone: geen build, geen dependencies, werken ook lokaal met een dubbelklik.
De assetbibliotheken zijn naslag/galerij met alle SVG's ingebed (geen losse bestanden nodig); de oefenbanken hebben hun visuals inline en linken nergens naar.

## Herkomst

Alle vragen zijn eigen parafrases van de examenstof — geen kopie van CBR-examenmateriaal.
Alle SVG's zijn eigen tekeningen. Niet-officieel oefenmateriaal.
